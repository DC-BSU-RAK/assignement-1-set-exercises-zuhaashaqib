HTML & CSS Quick Reference Guide

A quick guide to must-know HTML and CSS tools.

üìù Common HTML Tags

<h1> to <h6>

Defines headings. <h1> is the largest and <h6> is the smallest.

<h1>Welcome!</h1>

<p>

Defines a paragraph of text.

<p>This is a paragraph.</p>

<a>

Creates a hyperlink. Use target="_blank" to open in a new tab.

<a href="https://example.com" target="_blank">Visit Site</a>

<img>

Embeds an image.

<img src="image.jpg" alt="Description">

<ul>, <ol>, <li>

Creates unordered (bullets) and ordered (numbered) lists.

<ul>
  <li>Item 1</li>
  <li>Item 2</li>
</ul>

<div>

A container used for layout and grouping elements.

<div class="box">Content</div>

<span>

Used to style part of text inline.

<p>This is a <span class="highlight">highlight</span>.</p>

<nav>

Defines navigation links.

<nav>
  <a href="#html">HTML</a>
  <a href="#css">CSS</a>
</nav>

<footer>

Marks the footer of a document or section.

<footer>¬© 2025 My Website</footer>
